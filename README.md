# Проект

Описание проекта.

## Структура проекта

```
project-root/
├── index.html
├── about.html
├── css/
│   └── style.css
├── js/
│   ├── main.js
│   └── jquery.min.js
├── assets/
│   ├── images/
│   ├── icons/
│   └── fonts/
└── README.md
```

## Установка

1. Клонировать репозиторий
2. Открыть `index.html` в браузере

## Использование

Откройте `index.html` в любом современном браузере.
