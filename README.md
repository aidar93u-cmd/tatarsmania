## Запуск

Открыть любой `.html` в браузере. Сборка не требуется.

## Архитектура CSS

```
css/
├── lib/              # библиотеки (swiper, fancybox, aos)
├── style.css         # глобальные стили + общие компоненты
├── header.css        # шапка, мобильное меню, поиск
├── footer.css        # подвал
├── product-card.css  # карточки товаров
└── *.css             # стили конкретных страниц
```

## JS

```
js/
├── lib/              # библиотеки (jquery, swiper, fancybox, aos)
├── main.js           # общие функции
├── menu.js           # мобильное меню + поиск
└── *.js              # скрипты конкретных страниц
```
